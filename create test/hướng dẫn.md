* Để sinh test cần tạo 1 folder(dùng cho code:block)

\-b1: tạo 1 folder để chứ chương trình sinh test, và 2 file là code trâu có tỉ lệ đúng cao nhất, và 1 file là file chuẩn để chấm / nộp

\-b2: trong file chương trình sinh test chọn các kiểu RAND(l, r) để sinh số trong phạm vi l đến r

\-b3: chạy chương trình sinh test sinh từ 100 - 1000 test để kiểm tra ans giữa 2 file trâu và file chuẩn

\*NOTE: 1 file tên bài .cpp và 1 file tên bài \_trau.cpp



* Để tạo test chấm thì ta tạo thêm 1 file TESTS trong folder trên rồi chạy code sinh test là xong
* Có thể dùng code bằng template 03 để sinh test và compare
